Descargar Github

Descargar Git y Git Bash

Nuevo repositorio

Abre la terminal en el directorio del proyecto y usa

git init

para crear un nuevo repositorio de git.

Agregar archivos y hacer commit

Para registrar los cambios hechos puedes usar:

git add <archivo>

o

git add .

Luego, para hacer commit de estos cambios se usa

git commit -m "Mensaje"

Conectar el repositorio a GitHub

para conectar el repositorio local con GitHub se usa el comando

git remote add origin <server>

en <server> va a ir el link del repositorio, ejemplo:

https://github.com/usuario/repositorio.git

Enviar cambios al repositorio remoto

Para finalmente subir todos los cambios al repositorio se escribe

git push origin master

Este comando se utiliza cada vez que quieras actualizar cambios hechos y "commiteados" el repositorio en GitHub

Subir el proyeto a GitHub Pages

Una vez subido el proyecto, en la página del repositorio:

  1. Ir a settings
  2. Bajar la página y buscar la sección GitHub Pages
  3. En la sección Source elegir: master branch

Ahora GitHub va a generar un link parecido a este:

https://usuario.github.io/repositorio/

El cual va a ser el enlace directo a tu página
